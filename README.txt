# MRP – Media Ratings Platform
Small HTTP server in Java (no frameworks). Endpoints: `/api/users/register`, `/api/users/login`, …

## Run
- Java 21, Maven
- `mvn clean package`
- `java -jar target/<your-jar>.jar`
- Or `docker-compose up -d` (if applicable)

## API & Tests
- Postman collection: `postman/MRP_Postman_Collection.json`
- cURL demo: `scripts/demo.sh` / `scripts/demo.cmd`
